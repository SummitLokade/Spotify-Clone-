# Spotify Clone

A static frontend clone of Spotify’s web interface built using HTML and CSS. This project replicates the layout, styling, and interactive feel of the original platform — including hover effects and visual feedback.

## Features

- Sidebar with navigation: Home, Search, Your Library
- Playlist and podcast sections with clean layout
- Playback bar with timeline and volume slider (non-functional but visually responsive)
- Hover effects:
  - Light green glow on major buttons
  - Brightening effect on small icons and controls

## Tech Used

- HTML5
- CSS3
